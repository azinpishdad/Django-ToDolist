## Getting Started
### Prerequisites
* python 3.8
* django 3.1.6‍

### Installation
1. Clone the repo
‍‍ ‍‍`git clone  `
2. pip requirements
3. run django
`django-admin startproject myproject`
`python manage.py makemigrations app`
`python manage.py migrate`
